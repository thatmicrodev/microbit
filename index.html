02 About this board

This page is a hand-wired showcase for BBC micro:bit projects — hosted for free on GitHub Pages, no build step, no framework, no database. Every card above is a live GitHub Release from this repo, fetched straight from GitHub's API.

Want to add a new one? Draft a new Release on GitHub, attach your .hex file (or whatever you built), write a short description, publish. Nothing here needs editing — the card builds itself.
