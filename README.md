# 💻 About This Repository

This repository documents my programming journey as I continue learning, experimenting, and improving my skills. Every project represents a chance to explore a new concept, try a different programming technique, or simply create something fun.

Most of my work focuses on the **BBC Micro:bit**, taking advantage of its amazing built-in hardware, including:

* 💡 LED Display
* 🎮 Buttons
* 📱 Accelerometer
* 🧭 Compass
* 🌡️ Temperature Sensor
* 🔊 Sound
* 📡 Radio Communication
* 🔌 GPIO Pins

I enjoy seeing just how much can be created with such a small computer. Whether it's arcade-style games, puzzle games, utilities, animations, experiments, or hardware projects, I always try to push myself to build something new.
